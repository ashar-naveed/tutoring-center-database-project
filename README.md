# Tutoring Center Database Project

## Overview
Relational database designed for managing a tutoring center, including students, tutors, courses, sessions, and payments.

## Features
- Student and tutor management
- Course and session tracking
- Enrollment handling
- Payment records

## Tech Stack
- SQL (Oracle / MySQL — use what you actually used)
- ER Modeling

## Database Design
The database schema is designed using ER modeling and normalization principles.
See `er_diagram.png`.

## Files Included
- `schema_and_data.sql` – database schema and sample data
- `analysis_queries.sql` – SQL queries for data analysis and reporting
- `er_diagram.png` – ER diagram of the database
